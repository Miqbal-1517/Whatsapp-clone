# WhatsApp Clone - Real Time Chat Application

A real-time chat application built with Node.js and Socket.io, featuring WhatsApp-like interface.

## Features

- 💬 Real-time messaging
- 👥 Online users list
- ✍️ Typing indicators
- 🔒 Private messaging
- 📱 Fully responsive (Mobile friendly)
- 🎨 WhatsApp-like UI
- ⚡ Fast and lightweight

## Technologies Used

- **Backend**: Node.js, Express.js, Socket.io
- **Frontend**: HTML5, CSS3, JavaScript
- **Real-time Communication**: WebSockets (Socket.io)

## Installation

### Prerequisites

- npm (Node Package Manager)
